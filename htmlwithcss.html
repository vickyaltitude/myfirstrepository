Learning css

Ways of adding css to html

  1. Inline css
  2. Internal css
  3. External css

Inline css

Inline CSS is the one where styles are added inside the element itself

This element is styled using inline CSS



Internal CSS

Internal css is the one where styles are added inside the head using style element



External CSS

External CSS is the one where styles are added using exclusive css extension file

Preference rank for adding css with html

  1. Inline css (highest priority)
  2. Internal css
  3. External css (lowest priority)
